Конкурентные и именные лицензии на подключение

Выбирайте стартовую конфигурацию систему, отвечающую всем требованиям

Платформа Ivanti Service Manager имеет 3 варианта первоначальной конфигурации для приобретения: Help Desk, Service Desk и Service Management.

Варианты отличаются друг от друга набором процессов, доступных после установки:

    Пакет Help Desk включает в себя следующую базовую функциональность:

  • - Портал самообслуживания пользователей
  • - Управление инцидентами
  • - Управление запросами и Каталог услуг
  • - Управление знаниями
  • - Мобильный интерфейс и мобильное приложение
  • - Конструктор рабочих процессов
  • - Управление опросами пользователей
  • - Визуальные панели и отчётность

    Пакет Service Management включает в себя следующую базовую функциональность:

  • - Вся функциональность Service Desk
  • - Управление доступностью
  • - Управление проектами
  • - Управление релизами
  • - Управление событиями
  • - Управление финансами

    - Пакет Service Desk включает в себя следующую базовую функциональность:

  • --- Вся функциональность пакета Help Desk
  • --- Управление конфигурациями
  • --- Управление проблемами
  • --- Управление уровнем услуг (SLA)

Выбирая Ivanti Service Manager вы платите только за ту конфигурацию, модули которой используете. При повышении уровня зрелости и расширении используемого функционала вы просто обновляете лицензии до более высокого пакета. Благодаря тому, что минимальный пакет содержит только самый базовый функционал и стоит недорого, а функционал максимального пакета покрывает требования достаточно зрелых организаций, выбирая Ivanti Service Manager с самого начала, вы избегаете дорогостоящих миграций с системы на систему по мере развития.

Возможность подписки на использование системы при установке в собственной инфраструктуре

Для платформы Ivanti Service Manager доступна возможность взять лицензии «в аренду» - использовать лицензии на развернутый в собственной инфраструктуре Ivanti Service Manager по подписке, в течение оговоренного срока эксплуатации.

Каналы коммуникаций с пользователем

Ivanti Service Manager имеет отдельный пользовательский портал, позволяющий пользователям запрашивать стандартные услуги, регистрировать инциденты, просматривать свои ранее зарегистрированные заявки, пользоваться базой знаний через удобный и интуитивно понятный интерфейс портала самообслуживания.

Ivanti Service Manager имеет богатую функциональность по визуальному отображению информации в виде графиков и дашбордов, позволяющую оперативно контролировать происходящее в рамках автоматизируемых процессов и получать данные в различных разрезах.

Ivanti Service Manager имеет встроенную систему отчётности, основанную на технологиях Microsoft SQL Report Services и совместимую с ним. Просматривайте аналитические отчёты по процессам через интерфейс системы, без необходимости использовать отдельную внешнюю систему отчётности.

После установки Ivanti Service Manager вам сразу же будет доступен набор преднастроенных процессов, доступных для дальнейшей конфигурации или запуска в работу. Также присутствует мастер «горячей настройки» системы, позволяющий оперативно внести базовые настройки пользователей и минимальные изменения в преднастроенные рабочие процессы, и запустить систему в работу.

После установки Ivanti Service Manager предоставляет вам набор инструментов для организации каналов взаимодействия с пользователями – интегрируйтесь с сервером электронной почты, загрузите пользователей из службы каталогов и откройте для них портал самообслуживания. Также, можно интегрироваться с системой ip-телефонии Voice от Ivanti.

Платформа Ivanti Service Manager не имеет каких-либо «жёстких» рамок по настройке ITSM-процессов, изначально доступных в системе. Модифицируйте и развивайте эти процессы без ограничений, чтобы всегда соответствовать требованиям бизнеса.

Ivanti Service Manager позволяет создавать собственные рабочие процессы с нуля, используя платформу как конструктор бизнес-процессов. Для собственных процессов доступны все возможности настройки, которые есть в ITSM-процессах, изначально доступных в системе. Автоматизируйте деятельность бизнес-подразделений, информационной безопасности, управления складом и т.д.

Ivanti Service Manager легко интегрируется с другими информационными системами. Это помогает делать мастер настройки интеграционных подключений, позволяющий взаимодействовать Ivanti Service Manager со смежными системами, а также свой набор API для доступа к своим данным из других систем.

Для платформы Ivanti Service Manager имеет множество плагинов, как бесплатных, так и платных, доступных на Marketplace - https://marketplace.ivanticloud.com/. Эти плагины содержат в себе как готовые интеграционные коннекторы, позволяющие легко интегрироваться со смежными информационными системами, так и развитие функциональности самой платформы.
